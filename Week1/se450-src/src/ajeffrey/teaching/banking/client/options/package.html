Contains the options code for an SE 552 running example: a simple banking application.

This package contains an Options interface, implemented by OptionsImpl, which allows command-line options to be set for the client, which are then passed to the other classes. For example, a -debug flag can be put on the command line, which causes debugging information to be printed on the standard error.
